    Section 5.2 Probability distributions

    Some words.

    Subsection 5.2.1 Random variables and probability distributions

    See sections 3.1, 3.2. Recommended problems: (pg80)1,3,4ab,5,6,7a

    Definition 5.2.1. probability distribution.

    If \(\displaystyle X\) is a discrete random variable, the function given by \(\displaystyle f(x) = P(X = x)\) for each \(\displaystyle x\) within the range of \(\displaystyle X\) is called the probability distribution.

    Theorem 5.2.2. conditions for probability distribution.

    A function can serve as the probability distribution for a discrete random variable \(\displaystyle X\) if and only if its values, \(\displaystyle f(x)\text{,}\) satisfy the conditions
    1. \(\displaystyle f(x) \ge 0\) for each value within its domain;
    2. \(\displaystyle \sum_x f(x) = 1\text{,}\) where the summation extends over all the values within its domain

    Definition 5.2.3. distribution function.

    If \(\displaystyle X\) is a discrete random variable, the function given by where \(\displaystyle f(t)\) is the value of the probability distribution of \(\displaystyle X\) at \(\displaystyle t\text{,}\) is called the distribution function, or the cumulative distribution (function), of \(\displaystyle X\text{.}\)

    Theorem 5.2.4. properties of a distribution function.

    The values \(\displaystyle F(x)\) of the distribution function of a discrete random variable \(\displaystyle X\) satisfy the conditions
    1. \(\displaystyle F(-\infty) = 0\) and \(\displaystyle F(\infty) = 1\) (more carefully stated as \(\displaystyle \lim_{x\to-\infty}F(x) = 0\) and \(\displaystyle \lim_{x\to\infty} F(x) = 1\));
    2. if \(\displaystyle a \lt b\text{,}\) then \(\displaystyle F(a) \le F(b)\) for any real numbers \(\displaystyle a\) and \(\displaystyle b\)

    Exercises 5.2.2 Exercises

    1. Problem 3.1.

    For each of the following, determine whether the given values can serve as the values of a probability distribution of a random variable with the range \(x = 1, 2, 3, \text{ and } 4\text{:}\)
    1. \(\displaystyle f(1) = 0.25,f(2) = 0.75,f(3) = 0.25,\text{ and }f(4) = 0.25\)
    2. \(\displaystyle f(1) = 0.15,f(2) = 0.27,f(3) = 0.29,\text{ and }f(4) = 0.29\)
    3. \(\displaystyle f(1) = \dfrac{1}{19},f(2) = \dfrac{10}{19},f(3) = \dfrac{2}{19},\text{ and }f(4) = \dfrac{5}{19}\)

    2. Problem 3.3.

    Verify that \(\displaystyle f(x) = \dfrac{2x}{k(k+1)}\) for \(x=1, 2, 3, \dots, k\) can serve as the probability distribution of a random variable with the given range. Bonus: Find or describe \(F(x)\text{.}\)}

    3. Problem 3.4b.

    Determine \(c\) so that the function \(\displaystyle f(x) = c{5\choose x}\) for \(x = 0, 1, 2, 3, 4, 5\) can serve as a probability distribution of a random variable with the given range.

    4. Problem 3.5.

    For what values of \(k\) an
    \begin{equation*} f(x) = (1-k)k^x \end{equation*}
    serve as the values of the probability distribution of a random variable with countably infinite range \(x = 0, 1, 2, \dots\text{?}\)

    5. Problem 3.7a.

    Show that there are no values of \(c\) such that
    \begin{equation*} f(x) = \dfrac{c}{x} \end{equation*}
    can serve as the values of the probability distribution of a random variable with countably infinite range \(x = 0, 1, 2, \dots\)

    6. Problem 3.6.

    Construct a probability histogram for the probability distribution
    \begin{equation*} f(x) = \dfrac{{2\choose x}{4\choose{3-x}}}{{6\choose3}} \text{ for }x=0, 1, 2 \end{equation*}

    7. Problem 3.87.

    The probability distribution of \(V\text{,}\) the number of weekly accidents at a certain intersection, is given by \(g(0) = 0.4, g(1) = 0.3, g(2) = 0.2, g(3) = 0.1\text{.}\) Construct the distribution function of \(V\) and draw its graph.

    8. Problem 3.88.

    Using the statement and result of the previous problem, find the probability that there will be at least two accidents in any one week using
    1. the original probabilities, and
    2. the values of the distribution function.

    Subsection 5.2.3 Multivariate, marginal, and conditional distributions

    See sections 3.5, 3.6, 3.7. Recommended problems: (pg 101) 42, 44, 45, 49, 53, 54

    Definition 5.2.5. joint probability distribution.

    If \(\displaystyle X\) and \(\displaystyle Y\) are discrete random variables, the function given by \(\displaystyle f(x,y) = P(X=x, Y=y)\) for each pair of values \(\displaystyle (x,y)\) within the range of \(\displaystyle X\) and \(\displaystyle Y\) is called the joint probability distribution of \(\displaystyle X\) and \(\displaystyle Y\text{.}\)

    Theorem 5.2.6. conditions for a joint probability distribution.

    A bivariate function can serve as a joint probability distribution for a pair of discrete random variables \(\displaystyle X\) and \(\displaystyle Y\) if and only if its values, \(\displaystyle f(x, y)\text{,}\) satisfy the conditions
    1. \(\displaystyle f(x, y) \ge 0\) for each pair of values \(\displaystyle (x, y)\) within its domain;
    2. \(\displaystyle \sum_x\sum_y f(x, y) = 1\) where the double summation extends over all possible pairs \(\displaystyle (x, y)\text{.}\)

    Definition 5.2.7. joint distribution function.

    If \(\displaystyle X\) and \(\displaystyle Y\) are discrete random variables, the function given by
    \begin{equation*} F(x, y) = P(X \le x, Y \le y) = \sum_{s\le x} \sum_{t\le y} f(s, t) \text{ for } \infty \lt x, y \lt \infty \end{equation*}
    where \(\displaystyle f(s, t)\) is the value of the joint probability distribution of \(\displaystyle X\) and \(\displaystyle Y\) at \(\displaystyle (s, t)\text{,}\) is called the joint distribution function or joint cumulative distribution of \(\displaystyle X\) and \(\displaystyle Y\text{.}\)

    Definition 5.2.8. marginal distribution.

    If \(\displaystyle X\) and \(\displaystyle Y\) are discrete random variables and \(\displaystyle f(x, y)\) is the value of their joint probability distribution at \(\displaystyle (x, y)\text{,}\) the function given by
    \begin{equation*} g(x) = \sum_y f(x, y) \end{equation*}
    for each \(\displaystyle x\) within the range of \(\displaystyle X\) is called the marginal distribution of \(\displaystyle X\text{.}\) Correspondingly, the function given by
    \begin{equation*} h(y) = \sum_x f(x, y) \end{equation*}
    for each \(\displaystyle y\) within the range of \(\displaystyle Y\) is called the marginal distribution of \(\displaystyle Y\text{.}\)

    Definition 5.2.9. conditional distribution.

    conditional distribution
    \begin{equation*} f(x|y) = \dfrac{f(x, y)}{h(y)}, h(y)\ne 0 \end{equation*}
    \begin{equation*} w(y|x) = \dfrac{f(x, y)}{g(x)}, g(x)\ne 0 \end{equation*}

    Exercises 5.2.4 Exercises

    1. Problem 3.42.

    If the values of the joint probability distribution of \(X\) and \(Y\) are shown below,
    \begin{equation*} \begin{aligned}[t] P[x=0, y=0] \amp = \frac{1}{12}\\ P[x=1, y=0] \amp = \frac{1}{6}\\ P[x=2, y=0] \amp = \frac{1}{24}\\ P[x=0, y=1] \amp = \frac{1}{4}\\ P[x=1, y=1] \amp = \frac{1}{4}\\ P[x=2, y=1] \amp = \frac{1}{40}\\ P[x=0, y=2] \amp = \frac{1}{8}\\ P[x=1, y=2] \amp = \frac{1}{20}\\ P[x=0, y=3] \amp = \frac{1}{120}\\ \end{aligned} \end{equation*}
    find
    1. (b) \(P(X = 0, 1\le Y \lt 3)\)
    2. (c) \(P(X + Y \le 1)\)
    3. (d) \(P(X \gt Y)\)

    2. Problem 3.44.

    If the joint probability distribution of \(X\) and \(Y\) is given by
    \begin{equation*} f(x, y) = c(x^2+y^2) \text{ for } x=0, 3; y=0, 1, 2 \end{equation*}
    find the value of \(c\text{.}\)

    3. Problem 3.45.

    With references to the previous problem find
    1. \(\displaystyle P(X\le 1, Y \gt 2)\)
    2. \(\displaystyle P(X=0, Y\le 2)\)
    3. \(\displaystyle P(X +Y \gt 2)\)

    4. Problem 3.70.

    With reference to 3.42, find
    1. the marginal distribution of \(X\)
    2. the marginal distribution of \(X\)
    3. the conditional distribution of \(X\) given \(Y=1\)
    4. the conditional distribution of \(Y\) given \(X=0\)
