    Section 2.1 Expectation of continuous random variables

    This is the start of Chapter 4 in Freund‚Äôs Mathematical Statistics. Now we will study the major topics of this chapter with a focus on those applying to continuous random variables.

    Subsection 2.1.1 Expected value

    See section 4.1 - 4.2. Recommended problems: (pg 136) 7, 9, 10, 11, (pg 161) 60

    Definition 2.1.1. expected value (continuous).

    If \(X\) is a continuous random variable and \(f(x)\) is the value of its probability density at \(x\text{,}\) the expected value of \(X\) is given by
    \begin{equation*} E[X] = \int x\cdot f(x)\,dx \end{equation*}

    Example 2.1.2. expected value (continuous).

    Consider the probability density given below
    \begin{equation*} f(x) = \begin{cases}x, \amp \quad 0 \lt x \lt 1\\ 2-x, \amp \quad 1 \le x \lt 2\\ 0, \amp \quad \text{otherwise}\\ \end{cases} \end{equation*}
    Find \(E[X]\text{.}\)
    Solution.
    \begin{equation*} \begin{aligned}[t] E[X] \amp = \int_{-\infty}^\infty x\cdot f(x)\,dx\\ \amp = \int_{-\infty}^0 0\,dx + \int_{0}^1 \underbrace{x\cdot x}_{x^2}\,dx + \int_{1}^2 \underbrace{x\cdot(2-x)}_{2x-x^2}\,dx + \int_2^{\infty} 0\,dx\\ \amp = 0 + \int_{0}^1 x^2\,dx + \int_{1}^2 2x-x^2\,dx + 0\\ \amp = \Big(\dfrac{x^3}{3}\Big)\Big|_0^1 + \Big(x^2-\dfrac{x^3}{3}\Big)\Big|_1^2\\ \amp = \Big(\dfrac{1}{3} - 0\Big) + \Big(\Big(4-\dfrac{8}{3}\Big) - \Big(1-\dfrac{1}{3}\Big)\Big)\\ E[X] \amp = 1 \end{aligned} \end{equation*}

    Theorem 2.1.3. expected value of a function of a random variable (continuous).

    If \(X\) is a continuous random variable and \(f(x)\) is the value of its probability density at \(x\text{,}\) the expected value of \(g(X)\) is given by
    \begin{equation*} E[g(X)] = \int g(x)\cdot f(x)\,dx \end{equation*}
    Now would be a good time to review Theorem¬†4.1.3, Corollary¬†4.1.4, Corollary¬†4.1.5, and Theorem¬†4.1.7, all of which apply here as well. Recall, certain choices of \(g(x)\) in Theorem¬†2.1.3 give us the mean, variance, and other special moments.
    While we are at it, consider Theorem¬†4.1.3 which suggested that \(E[aX+b] = aE[X]+b\text{.}\) Defining expectation by integration gives a more convenient means of proving this statement.

    Proof.

    \begin{equation*} \begin{aligned}[t] E[aX+b] \amp = \int_{-\infty}^\infty (ax+b)\cdot f(x)\,dx\\ \amp = \int_{-\infty}^\infty ax\cdot f(x)\,dx + \int_{-\infty}^\infty b\cdot f(x)\,dx\\ \amp = a\int_{-\infty}^\infty x\cdot f(x)\,dx + b\int_{-\infty}^\infty f(x)\,dx\\ \amp = aE[x] + b \end{aligned} \end{equation*}
    Notice that the second of these integrals is \(1\) since \(f(x)\) is a probability density function.

    Subsection 2.1.3 Moments

    See section 4.3. Recommended problems: 4.3 (pg 146) 20, 22, 23, 31, 33, 34, 40, (pg 162) 69, 73, 75

    Definition 2.1.4. moments about the origin (continuous).

    The \(r^\text{th}\) moment about the origin of a random variable \(X\text{,}\) denoted by \(\mu_r'\text{,}\) is the expected value of \((X)^r\text{;}\) symbolically
    \begin{equation*} \mu_r'=E[(X)^r] = \int_{-\infty}^\infty x^r\cdot f(x)\,dx \end{equation*}
    for \(r = 0,1,2, \dots\) when \(X\) is continuous.

    Definition 2.1.5. moments about the mean (continuous).

    The \(\displaystyle r^\text{th}\) moment about the mean of a random variable \(X\text{,}\) denoted by \(\displaystyle \mu_r\text{,}\) is the expected value of \(\displaystyle (X-\mu)^r\text{;}\) symbolically
    \begin{equation*} \mu_r=E[(X-\mu)^r] = \int_{-\infty}^\infty (x-\mu)^r\cdot f(x)\,dx \end{equation*}
    for \(\displaystyle r = 0,1,2, \dots\) when \(X\) is discrete.

    Example 2.1.6. \(8^{\text{th}}\)-order polynomial density.

    Consider
    \begin{equation*} f(x) = \begin{cases}630x^4(1-x)^4, \amp \quad 0 \lt x \lt 1\\ 0, \amp \quad \text{otherwise}\end{cases} \end{equation*}
    Find \(\mu\) and \(\sigma^2\text{.}\)
    Hint.
    It is recommended that you expand the term \((1-x)^4\text{.}\) Doing that, and carefully integrating, you should find \(\mu = \sfrac{1}{2}\text{.}\) Additionally, by finding \(\mu_2' = E[X^2] = \sfrac{3}{11}\text{,}\) you will be able to use
    \begin{equation*} \sigma^2 = \mu_2' - (\mu)^2 \end{equation*}
    to find the variance is \(\sigma^2 = \sfrac{1}{44}\text{.}\)

    Subsection 2.1.5 Chebyshev‚Äôs Theorem

    See section 4.4.

    Theorem 2.1.7. Chebyshev‚Äôs Theorem.

    If \(\mu\) and \(\sigma\) are the mean and the standard deviation of a random variable \(X\text{,}\) then for any positive constant \(k\) the probability is at least \(1-\dfrac{1}{k^2}\) that \(X\) will take on a value within \(k\) standard deviations of the mean; symbolically
    \begin{equation*} P(|x-\mu| \lt k\sigma) \ge 1 - \dfrac{1}{k^2}, \sigma \ne 0 \end{equation*}
    Theorem¬†2.1.7 gives a lower bound on the probability that the value of the random variable is within a certain distance (specifically \(\pm k\sigma\)) of the mean.

    Exercises 2.1.6 Exercises

    2. at least \(0.99\text{?}\)

    Subsection 2.1.7 Moment-generating functions

    See section 4.5.

    Definition 2.1.8. moment-generating function (continuous).

    The moment-generating function of a random variable \(X\text{,}\) where it exists, is given by
    \begin{equation*} \displaystyle M_X(t) = E[e^{tX}] = \int_{-\infty}^\infty e^{tx}\cdot f(x)\,dx \end{equation*}
    when \(X\) is discrete.
    Notce that a moment-generating function \(\displaystyle M_X(t)\) itself is a function of the variable \(\displaystyle t\) not \(X\text{.}\) As it turns out, we are most interested in values of the function at or near \(t=0\text{.}\)
    Now would be a good time to review Theorem¬†4.1.14 and Theorem¬†4.1.16 which apply here in the continuous case as well.

    Example 2.1.9. moment-generating function by definition.

    Find the moment-generating function of
    \begin{equation*} f(x) = \begin{cases}e^{-x}, \amp \quad 0 \lt x \\ 0, \amp \quad \text{otherwise}\end{cases} \end{equation*}
    Solution.
    \begin{equation*} \begin{aligned}[t] M_X(t) \amp = E[e^{tX}]\\ \amp = \int_{0}^\infty e^{tx}\cdot e^{-x}\,dx\\ \amp = \int_{0}^\infty e^{-(1-t)x}\,dx\\ \amp = \lim_{A\to\infty}\Big(\Big(-\dfrac{1}{1-t}e^{-(1-t)x}\Big)\Big|_0^A\Big) \\ \amp = \lim_{A\to\infty}\Big(\Big(\underbrace{-\dfrac{1}{1-t}e^{-(1-t)A}}_{\text{needs }t \lt 1}\Big) - \Big(-\dfrac{1}{1-t}e^{-(1-t)0}\Big)\Big)\\ M_X(t) \amp = 0 + \dfrac{1}{1-t} \end{aligned} \end{equation*}
    For the first integral to converge we require \(t \lt 1\text{.}\)
    With the result from Example¬†2.1.9, assuming \(|t| \lt 1\) we can expand around \(t=0\) to find
    \begin{equation*} M_X(t) = \dfrac{1}{1-t} = 1 + t + t^2 + \dots + t^r + \dots \end{equation*}
    Remembering that \(\mu_r'\) is the coefficient of \(\dfrac{t^r}{r!}\text{,}\) we can rewrite the previous expansion as
    \begin{equation*} M_X(t) = \dfrac{1}{1-t} = 1 + 1\Big(\dfrac{t^1}{1!}\Big) + 2!\Big(\dfrac{t^2}{2!}\Big) + \dots + r!\Big(\dfrac{t^r}{r!}\Big) + \dots \end{equation*}
    which shows that \(\mu_r' = r!\text{.}\)
    Above the primary difficulty once the moment-generating function has been calculated is probably in the expansion of the Maclaurin series. One alternative to this requires calculating the integrals that define the necessary expectations
    \begin{equation*} \mu_r' = E[X^r] = \int_{-\infty}^\infty x^r e^{-x}\,dx \end{equation*}
    by integration by parts or tabular integration. Instead, given the moment-generating function, we could apply Theorem¬†4.1.14.

    Example 2.1.10. moments of an exponential by differentiation.

    Recall the moment-generating function from Example¬†2.1.9 was \(M_X(t) = \dfrac{1}{1-t}\) for \(t \lt 1\text{.}\) Find \(\mu_1'\) and \(\mu_2'\text{.}\)
    Solution.
    By theorem
    \begin{equation*} \begin{aligned}[t] \mu_1' \amp = \dfrac{d}{dt}\Big(M_X(t)\Big)\Big|_{t=0}\\ \amp = \dfrac{d}{dt}\Big(\dfrac{1}{1-t}\Big)\Big|_{t=0}\\ \amp = \Big(-(1-t)^{-2}(-1)\Big)\Big|_{t=0}\\ \mu_1' \amp = \Big(-(1-0)^{-2}(-1)\Big) = 1\\ \end{aligned} \end{equation*}
    To calculate \(\mu_2'\) we take the second derivative of \(M_X(t)\text{.}\) Starting from \(\dfrac{dM_X(t)}{dt} = (1-t)^{-2}\text{,}\) we find
    \begin{equation*} \begin{aligned}[t] \mu_2' \amp = \dfrac{d}{dt}\Big(\dfrac{dM_X(t)}{dt}\Big)\Big|_{t=0}\\ \amp = \dfrac{d}{dt}\Big((1-t)^{-2}\Big)\Big|_{t=0}\\ \amp = \Big(-2(1-t)^{-3}(-1)\Big)\Big|_{t=0}\\ \mu_2' \amp = \Big(-2(1-0)^{-3}(-1)\Big) = 2\\ \end{aligned} \end{equation*}
